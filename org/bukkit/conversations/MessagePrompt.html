MessagePrompt (org.bukkit.conversations.MessagePrompt)

MessagePrompt is the base class for any prompt that only displays a message to the user and requires no input.

Return Value Function
Prompt
messagePrompt:acceptInput(context [ConversationContext], input [String])

Accepts and ignores any user input, returning the next prompt in the prompt graph instead.
boolean
messagePrompt:blocksForInput(context [ConversationContext])

Message prompts never wait for user input before continuing.
Prompt
messagePrompt:getNextPrompt(context [ConversationContext])

Override this method to return the next prompt in the prompt graph.
